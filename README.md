# 🖼️ Galería SASS

Proyecto educativo que implementa una **galería de imágenes** utilizando únicamente **HTML** y **SASS (SCSS)**.  
El objetivo es demostrar cómo organizar estilos con variables, mixins y módulos para crear una galería visualmente atractiva y escalable.

---

## ✨ Características
- Galería responsiva construida con **HTML + SASS**.
- Uso de **variables** para colores y tipografías.
- **Mixins** para reutilizar estilos y simplificar código.
- Modularización con archivos parciales (`_variables.scss`, `_mixins.scss`, `_gallery.scss`).
- CSS compilado listo para producción (`main.css`).

---

## 📂 Estructura del proyecto

Galeria-SASS/
│── index.html       # Página principal con la galería
│── main.scss        # Archivo raíz de estilos SASS
│── main.css         # CSS compilado
│── main.css.map     # Mapa de compilación
│── _variables.scss  # Variables globales
│── _mixins.scss     # Mixins reutilizables
│── _gallery.scss    # Estilos específicos de la galería
│── img/             # Carpeta con imágenes de ejemplo


---

## 🛠️ Tecnologías utilizadas
- **HTML5**: estructura semántica de la galería.
- **SASS (SCSS)**: preprocesador CSS para modularidad y escalabilidad.
- **CSS3**: estilos compilados para el navegador.
